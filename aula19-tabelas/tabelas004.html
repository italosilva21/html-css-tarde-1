Tabelas Simples

tabelas sao um recurso muito importante para a organizacao dados no geral, em HTML para criar uma tabela usa-se a tang table e, dentro dela, alumas tangs de definição de conteudo, tais como:

  • tr ( Table Row)- linha da tabela
  • tr(Table Header)- cabeçalho da tabela
  • tr(Table Data)- dado da tabela

Uma tabela simples tem geralmente esta aparência

Codigo Nomes Dados
001 Alfredo Dados 1
002 Guilherme Dados 2
003 Larissa Dados 3

Anatonomia de Tabelas

Essa anatonomia é utilizada para tabelas grandes e melhora o resultados em sites de busca, consiste em adicionar os seguientes elementos na tabela:

  • caption- Legenda/ Descrição
  • Thead Initial-scale Cabeçalho
  • tbody -corpo
  • tfoot -Rodapé

    • todos essses elementos podem conter seus proprios tr , td e th( menos o caption)

    Independende da ordem que voce colocar as tangs da anatomia, o navegador sempre vai organizar na ortem correta.

    Scope

    É um atributo de um th que indica para qual direção o titulo estar mostrando os dados, pode ter os valores:

  • row - Dados em coluna
  • col - Dados em coluna
  • colgroup- Dados num conjunto de colunas( para celulas bmescladas)
  • rowgroup- Dados num conjunto de linhas ( para células mescladas)

    • Obs: Esses atributo é extremamente util para ajudar o site a ser relevante em sites de busca. Um exemplo de uma tebela utilizando todos os atributos da anatonomia citada a cima pode ser visto a seguir:

    Boletim
    Aluno Nota
    Ana 8
    Dimi 7
    Gustavo 10
    Lais 7
    Rodrigo 5
    Samira 6
    Vinicius 8
    Media 7,2

    Masclagem de celulas

    Mesclar celulas em uma tabela é algo bem comum e importante, em HTML, é necessario os seguintes atributos nas tags thou td;

  • colpan Mesclagem em coluna
  • rowspan

    • Depois, adiciona-se o valor que re4presenta o numero de celulas que ela irá ocupar. Um exemplo pode ser visto a seguir ;

    A B
    C D E
    F G

    Estilização de Colunas

    Para especificar uma coluna em uma tabela HTML, usa-se a tang colgroup, ela que define as colunas em uma tabela, cada coluna é representada por uma tag col que fica dentro da colgroup, isso facilita na hora de estilizar colunas com CSS

    A tag col possui o parametro span que especifica quantas colunas uma mesma tag col pode representar.
